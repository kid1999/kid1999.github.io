HDFS的配置和一些操作
发表于 2019-11-20| 更新于 2019-11-20|大数据
字数总计: 210|阅读时长: 1 分钟| 阅读量:

1.返回hadoop集群的信息：

hdfs dfsadmin -report

2.返回可访问的端口：

netstat -ntlp

3.hadoop的网页访问 ：

masterip + ：端口50070

4.关闭节点：

hadoop-daemon.sh stop namenode/datanode

5.hadoop/etc/hadoop 下的slaves管理着datanode：

vim slaves

填入所有 datanode名字（S1）

6.利用master管理所有子节点：

start-dfs.sh 启动所有节点

stop-dfs.sh 关闭所有节点

7.免密登陆：

ssh-keygen -t rsa
cd /.ssh

一路回车 生成两个文件 一个root的私钥 ，一个公钥

ssh-copy-id S1（子节点）

将公钥给子节点，以后就不用给密码了
